<HeaderRequestLink title="GET /api/events/{category}" />

Get events

### Query parameters:

| Header       | Description                  | Required |
|--------------|------------------------------|----------|
| category     | Events category filter. Available values: System, Clients, Queue, Deals | false |
| pageNumber   | Paging: position             | false    |
| pageSize     | Paging: returned items count | false    |

### Headers

| Header        | Value                    | Required |
|---------------|--------------------------|----------|
| Accept        | application/json         | true     |
| Authorization | Your authorization token | true     |
| Channel       | Your channel value       | true     |
| x-api-version | 2.0                      | true     |


## Request Example

```bash
curl -X GET "http://qrm1.kekker.com/api/events"
-H "accept: application/json"
-H "Authorization: {AUTHORIZATION}"
-H "channel: {CHANNEL}"
-H "x-api-version: 2.0"
```
