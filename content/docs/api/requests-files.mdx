<HeaderRequestLink title="POST /api/files" />

Upload file

### Query parameters:
none

### Body:

| Content Type        | Value            |
|---------------------|------------------|
| multipart/form-data | Form             |

### Headers

| Header        | Value                    | Required |
|---------------|--------------------------|----------|
| Accept        | application/json         | true     |
| Authorization | Your authorization token | true     |
| Channel       | Your channel value       | true     |
| x-api-version | 2.0                      | true     |


## Request Example

<TabBar>
    <TabItem label="Quorum">
        <CodeEditor code={
       `
curl -X POST "http://qrm1.kekker.com/api/files"
-H "accept: */*"
-H "Authorization: {AUTHORIZATION}"
-H "channel: {CHANNEL}"
-H "Content-Type: multipart/form-data"
-H "x-api-version: 2.0"
-F "formFile=offer.pdf;type=application/pdf"
       `
        } />
    </TabItem>
    <TabItem label="Hyperledger">

Download file

### Query parameters:

| Header       | Description                          | Type          |
|--------------|--------------------------------------|---------------|
| fileUid      | Unique local identifier for the file | string($uuid) |

### Headers

|---------------|--------------------------|----------|
| Accept        | application/json         | true     |
| Authorization | Your authorization token | true     |
